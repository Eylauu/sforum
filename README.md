# SForum
Un forum de discussion développé avec le framework [Symfony](https://symfony.com/).

## Installation
```
make
# affiche la liste de toutes les commandes

make install
# installe le projet et ses dépendances

make test 
# lance les tests

php bin/console server:run 
# lance le server
```
